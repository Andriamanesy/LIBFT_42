# LIBFT_42

This project is part of the 42 School curriculum and aims to recreate essential functions from the C standard library. It serves as a foundation for future projects by reinforcing memory management, string manipulation, and algorithmic logic in low-level programming.

🛠️ Key Features:
✅ Reimplementation of standard C functions (e.g., strlen, strcpy, malloc, etc.)
✅ Additional utility functions for string and memory operations
✅ Strict adherence to norminette coding standards
✅ Built-in unit tests for function validation

🚀 Objectives:
Strengthen understanding of C language fundamentals
Improve memory allocation and pointer manipulation skills
Develop a solid personal library for future projects

📂 Usage:
Compile the library using:

make

Then, link it with your own C programs:

gcc -L. -lft your_program.c -o your_program
